# 《陆地军事塔防》游戏项目

## 📋 项目概述

基于完整游戏设计文档（GDD）的**像素风2D Roguelike塔防游戏**，融合单位合并机制与经典塔防玩法。目标平台为**抖音小游戏**，支持碎片化时间游戏体验。

## 🎯 核心特色

- **Roguelike元素**：随机遗物系统，选择时暂停游戏
- **地块背包玩法**：土地**按单格购买**（一块块买）；升级后建筑占多格（如 Lv2 酒馆=田字形），需**手动指定**升级后占格才能合并
- **建筑合并**：同类型同等级合并，指定目标占格区域后升级，支持建筑移除
- **视野战斗**：单位自动发现视野内敌人/建筑并攻击，从下往上进攻
- **MVP 兵种**：步兵/坦克/火炮三兵种；码头、机场与海兵/空军后续扩展
- **地图扩展**：初始少量地块，单格购地扩展，摧毁敌方建筑解锁区域

## 🛠️ 技术栈

- **游戏引擎**：Godot 4.6.1
- **导出平台**：抖音小游戏（HTML5）
- **编程语言**：GDScript
- **美术工具**：AI生成SVG矢量图 + Aseprite
- **包大小**：主包≤4MB，整体≤20MB

## 🚀 快速开始
