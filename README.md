# 《陆地军事塔防》游戏项目

## 📋 项目概述

基于完整游戏设计文档（GDD）的**像素风2D Roguelike塔防游戏**，融合单位合并机制与经典塔防玩法。目标平台为**抖音小游戏**，支持碎片化时间游戏体验。

## 🎯 核心特色

- **Roguelike元素**：随机遗物系统，每次游戏体验不同
- **单位合并机制**：拖拽相同单位瞬间升级，视觉爆炸效果
- **自动战斗**：建筑产兵，单位自动推进，无需微操
- **军事主题**：现代陆地战场，坦克、步兵、火炮三兵种对抗

## 🛠️ 技术栈

- **游戏引擎**：Godot 4.6.1
- **导出平台**：抖音小游戏（HTML5）
- **编程语言**：GDScript
- **美术工具**：AI生成SVG矢量图 + Aseprite
- **包大小**：主包≤4MB，整体≤20MB

## 🚀 快速开始

1. 下载并安装 [Godot 4.6.1](https://godotengine.org/)
2. 下载 [抖音开发者工具](https://developer.open-douyin.com/)
3. 克隆项目并打开 `project.godot`
4. 参考 [开发计划与技术实现.md](开发计划与技术实现.md) 开始开发
